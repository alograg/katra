Katra 1.0

Estructura de directorios

  • /

    • docs: Documentacion del desarrollo

      • PHP: Documentacion de PHP
      • MySQL: Documentacion de Base de Datos
      • JS: Documentacion de JavaScript
      • CSS: Documentacion de Cascading Style Sheets
      • APIs: Documentacion de Interfaz de Programacion de Aplicaciones

    • lib: Contiene las librerias de PHP

      • js: Contiene librerias de JavaScript que se utilizan generalmente

      • css: Contiene librerias de CSS que se utilizan generalmente

      • php: Apartado para los archivos de PHP

        • externals: Contiele las librerias externas

          • AMFPHP
          • htmlparser
          • PHPMailer
          • reCaptcha
          • Twitter

        • katra: El framework de nuestros proyectos

          • applications: Contiene todas las aplicaciones disponibles.

            • cms: manejador de contenido.

          • database: Contenedor de los adaptadores a base de datos, abstracts e interface

            • dal: Capa de acceso a datos
            • mysql: Contiene los adaptadores para mysql

            • sqlite: Contiene los adaptadores para sqlite

                data: contiene los datos manejados por sqlite.
          • tools: Contiene clases utilitarias

    • web: Apartado para el contenido web

      • content: Carpeta de acceso publico, base del sitio

        • medias: Contenedora de archivos cargados

          • audios: Contiene solo archivos de audio (mp3,wav,aiff)

          • flash: Coniene solo archivos de flash (jpg,png,gif,swf)

          • docs: Coniene solo documentos (pdf,doc,xls,xml,ppt,txt,xslt,html)

          • files: Coniene solo archivos que no estan categorisados (zip,bin,tar,rar,*)

          • images: Coniene solo archivos de imagen (jpg,gif,png)

            • manipulated: Contiene imagenes manipuladas del directorio padre

          • videos: Coniene solo archivos de video (flv,3gp,avi,mov,wmv,mp4,f4v)

        • themes: Contiene look&feel, interacciones y servicios web exlusivos
          Pot cada tema consiste en una carpeta con esta estructura:

          • js: Contiene los archivos de JavaScript especificos del tema

          • languages: Contiene los archivos de lenguajes con los textos del tema

          • style: Contiene los archivos del deiseño

            • extras: Archivos de CSS para custiones especiales (IE,print,mobile)

            • img: Contiene los archivos de imagenes utilizadas en el diseño

          • templates: Contiene directorios para cada formato de archivo solicitado

            • boletin: Distribucion html para boletines

            • html: Distribucion html para navegadores

            • rss: Distribucion xml para lectores de RSS

          • ws: Contiene archivos que suministran servicios especiales al tema
