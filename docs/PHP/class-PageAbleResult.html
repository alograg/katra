Class PageAbleResult

PageAbleResult is an AMFPHP service class which is used internally by AMFPHP to provide support for pageable recordsets. The methods of PageAbleResult are called automatically by the Flash player when implementing pageable recordsets. To use pageable recordsets the developer need only include the pagesize value in the service class method table and use setDeliveryMode in the Flash client.
