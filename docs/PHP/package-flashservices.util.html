Package flashservices\util

Classes summary
AMFObject AMFObject is a datatype representing the parsed representation of the binary AMF data.
Authenticate The Authenticate package is used to define helper methods related to authentication.
DateWrapper The DateWrapper allows easy handling of Flash dates
DescribeService The DescribeService is used to provide a description of the class to the service browser
Headers The Headers class includes a static method getHeader available from all services that allows one to get an AMF header from any service like the Flash one
MessageBody AMFBody is a data type that encapsulates all of the various properties a body object can have.
MessageHeader AMFHeader is a data type that represents a single header passed via amf
NetDebug The NetDebug class includes a NetDebug::trace function that works like the Flash one
